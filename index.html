RemoteShield Tool Testing
PlayOn High School

Pixellot has given you early access to a remote tool — RemoteShield — that can configure the network, firewall, and content filter settings of a Pixellot VPU from anywhere, as long as the device is powered on. You have been assigned to test it on a brand-new installation. You will report on your findings, methodology, and protocols to introduce this tool to the Senior Specialists. Treat this case study as close to a real scenario as possible. Note: External tools might not reflect the information given in this case study.

Completion Checklist

  • Pixellot VPU is properly configured for all network settings.
  • Communication with PlayOn High School.
  • Findings & feedback on RemoteShield.
  • Plan to implement the tool for the rest of SST.
  • Present the Case Study to the Hiring Manager(s).

What You Have Access To

  • RemoteShield FW-2000 — full read/write access via the RemoteShield tab
  • Ethernet Settings — configure IP addressing mode and network parameters
  • Firewall Policies — manage outbound port rules for the Pixellot VPU
  • Content Filter — manage domain whitelist entries
  • Advanced Settings — review security and performance settings

Note

This is a brand-new installation. No Pixellot-specific configuration has been applied yet.
